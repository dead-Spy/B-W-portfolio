
# 🌘 Tanvir Amin | Black & White Portfolio

A minimalist, high-contrast digital experience showcasing the intersection of professional illustration and full-stack engineering. Built with a focus on typography, motion, and a monochromatic aesthetic.

---

## 🌑 The Aesthetic
This portfolio embraces a **monochromatic design language**, stripping away the noise of color to focus purely on form, composition, and interaction. 

- **Theme:** High-Contrast Black & White
- **Focus:** Digital Illustration & Animation
- **Experience:** Fluid, smooth-scrolling, and immersive

## 🛠 Tech Stack (The Core)

- **Framework:** Vue.js 3 (Composition API)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 (Monochrome Palette)
- **Animations:** GSAP & Lenis (Smooth Scroll)
- **Build Tool:** Vite 6
- **Asset Handling:** Vue3 Lottie

## ✨ Key Features

- **Monochrome UI:** A carefully crafted B&W interface for a timeless feel.
- **GSAP Interactions:** Sophisticated, scroll-triggered animations.
- **Optimized Performance:** Built with the latest Vite 6 engine for near-instant load times.
- **Type Safety:** Robust development with TypeScript.

## 🚀 Local Setup

\`\`\`bash
# Clone the repository
git clone https://github.com/dead-Spy/illustration-portfolio.git

# Enter directory
cd "Portfolio -B&W"

# Install dependencies
npm install

# Start development
npm run dev
\`\`\`
